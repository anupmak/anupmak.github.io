My Work

I worked on multiple projects which gave me wonderful learning opportunities and exposure to various technologies. I would like to share few of them here. Please feel free to have a look and provide your feedback through emails. I appreciate your inputs.

WebsiteMaker

This project create Websites without any coding knowledge. This works well for both mobile devices and the traditional websites. The user can add new websites, pages and widgets to the page. These widgets then can be sorted by simply dragging and droping.

Sociali

Project finds user location and suggest events around them. Events are picked within a distance of given radius. User preferences for event type is stored, further vents are picked accordingly and are displayed on google maps.

Sentiment Analysis

Project implemented add-1 (Laplace) smoothing and classification. For this classification I implemented Naive Bayes text categorization system to predict whether movie reviews are positive or negative. This is categorization called “sentiment analysis”. This can help companies with the feedback.
